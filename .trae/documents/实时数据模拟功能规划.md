# 实时数据运行模拟功能规划

## 功能目标

针对数据表展开实时数据运行模拟，将 AI 分析发现的问题特征在模拟中体现，帮助用户更直观地理解数据问题和改进效果。

## 核心功能模块

### 1. 数据模拟引擎
- **定时数据生成**：按时间间隔自动生成模拟数据
- **参数化模拟**：支持根据 AI 分析的特征进行参数调整
- **异常模拟**：模拟 AI 发现的异常情况（缺失值、异常值、数据漂移等）

### 2. AI 分析特征映射
- **问题特征提取**：从 AI 分析结果中提取关键特征
  - 数据缺失率高的字段
  - 存在异常值的字段
  - 数据分布不均的字段
  - 关键指标识别
- **模拟参数映射**：将问题特征转换为模拟参数

### 3. 实时监控面板
- **数据流展示**：实时显示数据变化
- **指标监控**：关键指标实时更新
- **异常告警**：模拟过程中发现问题实时提示

### 4. 对比分析
- **原始数据 vs 模拟数据**：对比展示
- **改进前后对比**：展示优化效果

---

## 技术方案

### 后端
- 新增 `/api/v1/simulation` 相关接口
- 使用 Python 定时任务（如 APScheduler）生成模拟数据
- 支持 WebSocket 实时推送数据变化

### 前端
- 新增"数据模拟"标签页
- 使用 ECharts 或其他图表库展示实时数据
- 添加控制面板（开始/停止/参数调整）

## 实施步骤

2. **后端接口开发**：创建模拟引擎 API
